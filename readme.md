# Русификатор для Sludge Life: The Bit Mud Sessions

Данный репозиторий содержит 2 файла, которые русифицируют **Sludge Life: The Bit Mud Sessions**.

## Инструкция по установке

1. Скачайте два файла из этого репозитория:
   * `resources.assets`
   * `sharedassets0.assets`
2. Переместите скачанные файлы в корневую папку установленной игры (обычно в папку `Sludge Life_Data`, где находятся оригинальные файлы `.assets`).
3. Подтвердите замену файлов в целевой папке.

## Активация русского языка

После успешной замены файлов выполните следующие действия:
1. Запустите игру.
2. Перейдите в настройки языка.
3. Выберите **португальский язык** (Portuguese) — вместо него в игре будет отображаться **русский**.
